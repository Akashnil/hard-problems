**Problem 6:**

For a graph $G=(V,E)$, let $G_{S}=(V,E(S,S))$ denote the graph with the same vertex set, but only the edges between vertices in S. Let L be the Laplacian matrix of G and let $L_{S}$ be the Laplacian of $G_{S}$. I say that a set of vertices S is e-light if the matrix $\epsilon L-L_{S}$ is positive semidefinite. Does there exist a constant $c>0$ so that for every graph G and every between 0 and 1, V contains an e-light subset S of size at least $c\epsilon|V|$?
